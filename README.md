OTT Content Analytics Dashboard (Power BI)
Project Overview

This project presents an interactive Power BI dashboard designed to analyze content across major OTT platforms including Netflix, Amazon Prime Video, and Disney+ Hotstar.

The goal of this project is to explore and visualize content distribution, genre popularity, ratings, duration patterns, and yearly release trends across different platforms to understand OTT content strategies.

The dashboard enables users to interactively filter and explore data through slicers and platform-specific pages.

Objectives

The main objectives of this project were:

Analyze the distribution of Movies vs TV Shows across OTT platforms.

Identify popular genres and content categories.

Examine content ratings and audience classification.

Understand yearly trends in content releases.

Compare content strategies across Netflix, Amazon Prime, and Disney+ Hotstar.

Tools & Technologies Used

Power BI – Data visualization and dashboard creation
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Creating calculated measures and KPIs
Data Modeling – Star schema design for efficient analysis

Data Cleaning & Transformation

Data preprocessing was performed using Power Query Editor to ensure clean and structured data for analysis.

Key transformations included:

Removing unnecessary columns

Removing duplicate records

Splitting multi-value columns such as Country and Genre

Converting comma-separated values into individual rows

Creating normalized tables for efficient relationships

Preparing data for star schema modeling
